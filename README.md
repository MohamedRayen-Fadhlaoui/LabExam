# Tunisia Heritage Quest 🏛️

**Developed by: Mohamed Rayen Fadhlaoui**

Tunisia Heritage Quest is a Mediterranean-themed Android application built with **Jetpack Compose**. It challenges users to identify famous Tunisian monuments and cities through high-quality images, celebrating over 3,000 years of history.

## Features
- **Identify from Image**: Core quiz mechanic focused on visual recognition.
- **90 Questions**: Extensive database covering Roman, Punic, Islamic, and Modern Tunisian history.
- **Difficulty Levels**: Easy, Medium, and Hard modes.
- **Mediterranean UI**: Custom theme using `MedBlue`, `TerraCotta`, and `Beige` palettes.
- **MVVM Architecture**: Clean and maintainable code structure.

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **Navigation**: Compose Navigation
- **Image Loading**: Local Resource Mapping

## How to Run
1. Clone the repository.
2. Open in Android Studio (Ladybug or newer).
3. Build and Run on an Emulator or Physical Device.
